📦 InstagramBot Quraşdırma və İstifadə Təlimatı

1. Lazım olan paketləri quraşdır:
   CMD aç və yaz:
   py -m pip install -r requirements.txt

2. Cookie faylı çıxartmaq üçün:
   CMD-də yaz:
   py get_cookies.py
   ✅ Bu, avtomatik olaraq cookies.txt faylını yaradacaq

3. Botu işə salmaq üçün:
   CMD-də yaz:
   py bot.py

4. Sonra Telegram-da botunuza Instagram linki göndərin:
   Məsələn: https://www.instagram.com/p/C8TG6N1sWFn/

🔁 Bot sizə media faylını (video/jpg) geri göndərəcək.
